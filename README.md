# Flipkart Login Page Clone

This project is a simple clone of the Flipkart login page created to practice HTML and CSS fundamentals.

It was one of my first frontend projects and helped me understand page layouts, CSS styling, positioning, and responsive design concepts.

---

## Features

- Flipkart-inspired login page
- Clean user interface
- Responsive layout
- HTML5 structure
- CSS3 styling
